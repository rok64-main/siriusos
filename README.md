SiriusOS
SiriusOS is a fast, web-based operating system designed to run entirely inside a single browser tab. It provides a private workspace with a clean, modern look that works on almost any network.

Key Features
Desktop Interface: A full desktop with windows that you can move, resize, and layer. It includes a taskbar to track your open apps and a start menu for quick access.

Privacy Tools:

Tab Cloak: This tool lets you "disguise" the page. It can change the tab's name and icon to look like a normal site (such as Google Drive) so it doesn't stand out in your browser history.

Proxy Gateway: A built-in window that lets you browse other websites safely inside SiriusOS.

Built-in Apps:

Sirius AI: A smart assistant for answering questions or helping with projects.

Paint: A drawing app with different colors and brush sizes.

Notes: A simple text editor that automatically saves your typing so you don't lose your work.

Weather: A live tracker that shows the current temperature and conditions.

Design and Customization
Visual Themes: You can instantly change the look of the OS with themes like Midnight, Sunset, Forest, and Dark Slate.

Live Stats: The taskbar shows a live clock with seconds and a battery meter that matches your actual device's power level.

Smooth Animations: Every window and menu opens with smooth transitions to make the experience feel like a real computer.
